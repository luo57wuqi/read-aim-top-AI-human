# read-aim-top-AI-human
# 📖 极简智读 (Smart Reader) v2.5

> **沉浸式 · 本地化 · AI 驱动**
> 
> 一个单文件、无后端、支持深度阅读与思考的 Markdown 阅读器。

![License](https://img.shields.io/badge/license-MIT-blue.svg) ![HTML5](https://img.shields.io/badge/HTML5-Single%20File-orange.svg) ![AI](https://img.shields.io/badge/AI-Powered-purple.svg)

**极简智读** 是一款专为深度阅读者设计的浏览器端应用。它无需安装、无需联网（基础阅读功能），支持 Markdown 文件导入。通过内置的 AI 助手、思维导图和数据看板，帮助你不仅“读完”一本书，更能“读懂”一本书。

## ✨ 核心亮点

### 1. 沉浸式阅读体验
*   **多主题切换**：内置羊皮纸、面壁灰、香芋紫、护眼绿、暗黑模式等 6 种配色。
*   **排版优化**：自动生成侧边栏目录，支持字号无级调节，专注模式自动隐藏 UI。
*   **单文件运行**：所有代码集成在一个 `.html` 文件中，双击即用，随身携带。

### 2. 深度思考工具
*   **划线与笔记**：支持高亮划线，并添加批注想法。
*   **全书思维笔记**：独立的宏观笔记编辑区，支持自动记录日期日志，随时记录稍纵即逝的灵感。
*   **思维导图可视化**：一键生成全书脉络图，将目录、章节总结、高亮笔记自动转化为思维导图。

### 3. AI 智能阅读助手 (基于通义千问)
*   **划线提问**：选中文字，一键让 AI 解释、翻译、批判或总结。
*   **自定义预设**：内置“解释”、“总结”、“翻译”、“提问”等快捷指令，支持用户自定义 Prompt。
*   **全书分析**：支持对选中的多条笔记进行 AI 综合深度分析。

### 4. 数据化阅读痕迹
*   **数据看板**：可视化展示阅读进度、总时长、笔记数量及每日阅读活跃度图表。
*   **行为日志 (Log)**：全自动记录你的每一次阅读、翻页、笔记和修改行为，形成时间轴。
*   **数据导出**：支持导出为 `.md` 文件，**完美保留**阅读进度、高亮、笔记和日志数据，方便在不同设备间同步。

## 🚀 快速开始

### 安装
本项目**无需安装**，也无需配置 Node.js 环境。
1. 下载本项目中的 `index.html` 文件（或 `read.html`）。
2. 直接使用 Chrome / Edge / Safari 等现代浏览器打开即可。

### 导入书籍
1. 点击右上角的 **“导入”** 按钮。
2. 选择本地的 `.md` (Markdown) 文件。
3. *可选*：输入你的阿里云 DashScope API Key 以启用 AI 功能。

---

## 🤖 AI 配置指南

本项目默认接入阿里云 **通义千问 (Qwen-Turbo)** 模型。

1. 前往 [阿里云百炼控制台](https://bailian.console.aliyun.com/) 申请 API Key。
2. 在软件“导入”弹窗中填入 Key（Key 会保存在本地浏览器缓存中，不会上传服务器）。
3. **自定义 AI 角色**：
    * 打开 AI 助手面板，点击顶部的 ⚙️ 图标。
    * 你可以修改系统提示词（System Prompt）。
    * 也可以直接点击输入框上方的快捷标签（解释、翻译、总结等）。

## 🔒 隐私与数据说明

*   **Local-First (本地优先)**：所有的书籍内容、笔记、进度均存储在你的浏览器内存中，**不会上传**到任何服务器。
*   **AI 数据**：只有当你主动使用 AI 功能（如划线提问）时，选中的文本段落会被发送至阿里云 API 进行处理。
*   **备份**：建议定期点击右侧边栏的 **“导出数据”** 按钮，将带有阅读数据的 `.md` 文件保存到本地备份。
